# popcorn
복이방 react + spring boot(api server)미니 프로젝트

- 폴더 구분 방법
 - 주요하게 사용되는 component 폴더, js파일을 만듦
 - 공통되는 내용이 있다면 commons에 생성하기.
 
- 파일 명 작성 규칙
 > camelCase > 첫 단어 소문자 두 번째 단어부터 첫 글자 대문자로
 > PascalCase > 첫 단어도 대문자로 시작.
 - 폴더 : camelCase
 - 파일 : camelCase
 - 컴포넌트 : PascalCase
 - 함수 : camelCase
 - 변수 : camelCase
